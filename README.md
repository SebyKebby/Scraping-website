
# Scraping-website 

Using selenium, a template for scraping a list of 1500 luxury hotels

## Lessons Learned
Mastering using selenium to scrape sites that stop you from scraping it with Scrapy.
## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
